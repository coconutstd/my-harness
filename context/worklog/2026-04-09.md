## 0cd42b0 — Add Worklog, Branching, Task Management harness

### 맥락
3개 신규 하네스를 추가했다.
- Worklog: 커밋 단위로 작업 맥락 기록. 커밋이 없으면 기록도 없다는 원칙 채택.
- Branching: GitHub Flow 규칙을 에이전트에게 전달. strategy.md(정적)와 active-branches.md(동적) 분리.
- Tasks: 사용자가 자유 형식으로 요구사항 제시 → 에이전트가 구조화하여 TASK-NNN.md로 적재.
